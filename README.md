# Website Mầm Non (WordPress Project)

## Mô tả

Dự án xây dựng website cho trường mầm non, gồm các chức năng:

- Trang chủ giới thiệu
- Tin tức
- Đăng ký tuyển sinh
- Trang liên hệ
- Quản lý bài viết qua WordPress

## Công nghệ

- WordPress 6.x
- PHP 8.x
- MySQL
- Giao diện tùy chỉnh trong wp-content/themes/

## Hướng dẫn cài đặt

1. Clone project:
   git clone https://github.com/tenban/mam-non-wordpress.git

2. Tạo database mới:
   Import file database từ thư mục /database (nếu có)

3. Copy wp-config-sample.php → đổi tên thành wp-config.php  
   Điền thông tin database

4. Chạy bằng XAMPP hoặc Laragon
